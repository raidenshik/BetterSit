# BetterSit 🪑

BetterSit — это простой плагин для Paper 1.21.11, который позволяет игрокам садиться командой `/sit`.
Плагин поддерживает настройку через конфиг, мультиязычность (MiniMessage + HEX) и автоматическое вставание при нажатии Shift.

## Возможности
- Команда `/sit` для посадки на месте.
- Автоматическое вставание при нажатии Shift.
- Поддержка нескольких языков (en, ru и другие).
- Настраиваемое смещение по высоте и центрирование по блоку.
- Кулдаун на использование команды.
- Перезагрузка конфигурации через `/bettersit`.

## Установка
1. Скачайте последнюю версию плагина из [Releases](https://github.com/raidenshik/BetterSit/releases).
2. Поместите JAR-файл в папку `plugins` вашего сервера Paper 1.21.11.
3. Перезапустите сервер (или используйте `/reload`, хотя это не рекомендуется).
4. Готово! Игроки могут использовать `/sit`.

## Команды
| Команда | Описание | Право |
|---------|----------|-------|
| `/sit` | Сесть на месте. | `betterSit.sit` |
| `/bettersit` / `bsit` | Перезагрузить конфигурацию и языки. | `betterSit.admin` |

## Права (Permissions)
- `betterSit.sit` — доступ к команде `/sit` (по умолчанию `true`).
- `betterSit.admin` — доступ к команде `/bettersit` (по умолчанию `op`).

## Конфигурация (`config.yml`)
```yaml
# Язык сообщений (en, ru или другой файл в папке languages)
language: en

# Центрировать ли игрока по центру блока?
center-on-block: true

# Смещение по высоте (отрицательное значение опускает)
sit-height-offset: -0.2

# Задержка между использованиями /sit (секунды, 0 - отключено)
sit-cooldown: 0
